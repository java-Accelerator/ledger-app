## 项目

### 记账 Web 应用（主项目）

完整的个人记账工具，前后端分离，多用户，生产环境运行中。

- 仓库：https://github.com/java-Accelerator/ledger-app
- 在线：http://124.222.28.100:8080
- 本地开发：`python -m uvicorn api_ledger:app --reload --port 8000`，前端 `web_ledger.html` 用浏览器打开
- 测试：`pytest test_api_ledger.py -v`（11 用例，需设环境变量 `LEDGER_DB=test_ledger.db`）

功能：注册/登录（token + 密码哈希）、记收支、按类型筛选、删除、本月统计、分类条形图、多用户数据隔离。

技术栈：

| 本地开发 | 生产部署 |
|---------|---------|
| FastAPI + uvicorn | Flask + Gunicorn（内置） |
| SQLite (ledger.db) | 同 SQLite |
| 原生 HTML/CSS/JS | 同前端 |

架构（生产环境）：

```
浏览器 → 124.222.28.100:8080 → Nginx → 127.0.0.1:5000 → Flask
```
