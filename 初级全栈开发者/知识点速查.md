# 全课程知识点速查

## Python 基础

| 概念 | 用法 | 示例 |
|------|------|------|
| 变量 | 给数据起名字 | `age = 25` |
| 类型转换 | 不同数据类型之间切换 | `int("42")`, `str(100)`, `float(input())` |
| f-string | 把变量嵌进字符串 | `f"你好{name}，{score}分"` |
| if/elif/else | 条件判断 | `if score >= 90: ... elif score >= 60: ... else: ...` |
| while | 条件为真就反复执行 | `while True: ... break` |
| for | 遍历列表或序列 | `for s in students: print(s["name"])` |
| list | 有序集合，可增删改 | `scores = [85, 92, 78]`, `scores.append(88)` |
| dict | 键值对，通过名字取值 | `{"name": "小明", "score": 85}` |
| def | 定义函数，封装可复用代码 | `def add(a, b): return a + b` |
| return | 函数输出结果 | `return total` |
| import | 引入模块 | `import random`, `from fastapi import FastAPI` |
| try/except | 捕获异常防止崩溃 | `try: int(x) except ValueError: ...` |
| `==` vs `=` | 比较 vs 赋值 | `if a == 5:` vs `a = 5` |

## Python 进阶

| 概念 | 用法 | 示例 |
|------|------|------|
| enumerate | 遍历同时获取索引 | `for i, item in enumerate(items, 1):` |
| list.pop | 按位置删除并返回 | `todos.pop(index)` |
| f-string `:.1f` | 控制小数位数 | `f"{avg:.1f}"` |
| random.randint | 生成随机整数 | `random.randint(1, 100)` |
| os.path.exists | 判断文件是否存在 | `os.path.exists("data.txt")` |
| os.environ.get | 读环境变量，有默认值 | `os.environ.get("LEDGER_DB", "ledger.db")` |
| secrets.token_hex | 加密安全随机字符串 | `secrets.token_hex(32)` |
| hashlib.pbkdf2_hmac | 密码哈希（不可逆） | `hashlib.pbkdf2_hmac("sha256", pw, salt, 100000)` |

## HTML / CSS / JS

| 概念 | 用法 | 示例 |
|------|------|------|
| HTML 标签 | 标记内容 | `<h1>`, `<p>`, `<button>`, `<input>` |

## 后端 (FastAPI / Flask)

| 概念 | 用法 | 示例 |
|------|------|------|
| @app.get / @app.post | 路由装饰器 | `@app.get("/todos")` |
| {参数} 路径参数 | URL 中的动态值 | `/records/{record_id}` |
| Query 参数 | URL ? 后面的筛选 | `?type=收入` |
| Header 参数 | HTTP 请求头中取值 | `Authorization: Bearer xxx` |
| Depends | 依赖注入，自动鉴权 | `user: dict = Depends(get_current_user)` |
| HTTPException | 返回错误状态码 | `raise HTTPException(status_code=401)` |
| CORS 中间件 | 允许跨域请求 | `CORSMiddleware(allow_origins=["*"])` |
| BaseModel | 定义数据形状 | `class Record(BaseModel): date: str` |
| jsonify (Flask) | 返回 JSON 响应 | `return jsonify({"ok": True})` |
| send_file (Flask) | 返回静态文件 | `return send_file("index.html")` |

## 数据库 (SQLite)

| 概念 | 用法 | 示例 |
|------|------|------|
| CREATE TABLE | 建表 | `CREATE TABLE IF NOT EXISTS records (...)` |
| INSERT INTO | 插入数据 | `INSERT INTO records (...) VALUES (?, ?, ?)` |
| SELECT | 查询数据 | `SELECT * FROM records WHERE user_id = ?` |
| UPDATE | 修改数据 | `UPDATE users SET token = ? WHERE id = ?` |
| DELETE | 删除数据 | `DELETE FROM records WHERE id = ?` |
| WHERE | 条件过滤 | `WHERE type = '收入' AND user_id = ?` |
| ORDER BY | 排序 | `ORDER BY date DESC` |
| GROUP BY + SUM | 分组统计 | `SELECT type, SUM(amount) GROUP BY type` |
| `?` 占位符 | 防 SQL 注入 | `conn.execute("... WHERE id = ?", (id,))` |
| commit | 写操作后提交 | `conn.commit()` |
| lastrowid | 刚插入行的 id | `cur.lastrowid` |
| PRAGMA busy_timeout | 锁等待超时 | `PRAGMA busy_timeout=5000` |

## 鉴权与安全

| 概念 | 用法 | 示例 |
|------|------|------|
| 密码哈希 | 库中不存明文 | `hashlib.pbkdf2_hmac(...)` |
| 随机盐 | 每次哈希加随机字符 | `salt = secrets.token_hex(16)` |
| Token | 登录后签发的临时凭证 | `Authorization: Bearer <token>` |
| 多用户隔离 | 每张表加 user_id | `WHERE user_id = ?` |
