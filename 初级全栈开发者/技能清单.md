# 技能清单

## 编程语言

- **Python** — 熟练掌握，能独立完成从数据建模到 API 开发的完整后端链路

## 后端开发

- **FastAPI** — 路由设计、路径参数、Query 参数、依赖注入鉴权、CORS 中间件、Pydantic 数据模型
- **Flask** — 路由、请求处理、JSON 响应、静态文件服务、蓝图结构
- **RESTful API** — 完整 CRUD 设计，GET/POST/PUT/DELETE，状态码规范

## 数据库

- **SQLite** — 表设计、CRUD 操作、分组统计、多用户数据隔离、并发锁处理
- **SQL** — SELECT/INSERT/UPDATE/DELETE/WHERE/ORDER BY/GROUP BY/参数化查询防注入

## 前端

- **HTML** — 语义化标签、表单、页面骨架
- **CSS** — Flexbox 布局、选择器、响应式设计、暗色模式
- **JavaScript** — DOM 操作、事件监听、fetch 异步请求、localStorage 持久化、前后端联调

## 鉴权与安全

- 密码哈希（hashlib.pbkdf2_hmac + 随机盐）
- Token 认证（secrets.token_hex）
- 多用户数据隔离（user_id 字段 + WHERE 过滤）
- HTTP 状态码鉴权（401 Unauthorized）

## 测试

- **pytest** — 单元测试、接口测试、fixture 隔离
- **TestClient** — FastAPI 接口模拟测试

## 工程化

- Git 版本控制（init/add/commit/push/.gitignore）
- requirements.txt 依赖管理
- README.md 项目文档
- 环境变量配置管理

## 部署与运维

- 腾讯云服务器部署
- Nginx 反向代理配置
- 安全组/防火墙端口管理
- nohup 后台进程管理
- scp 远程文件传输

## 项目经验

### 个人记账 Web 应用

前后端分离的在线记账工具，支持注册登录、多用户数据隔离、收支记录管理、本月统计和分类图表。

- 后端：FastAPI（本地开发）/ Flask（生产部署）
- 前端：原生 HTML + CSS + JavaScript
- 数据库：SQLite
- 测试：pytest 11 用例，覆盖率包括注册、登录、鉴权、CRUD、统计
- 部署：腾讯云 + Nginx 反向代理
